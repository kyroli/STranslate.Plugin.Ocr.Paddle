# STranslate.Plugin.Ocr.Paddle

基于 OpenVINO 的 STranslate PaddleOCR 离线文本识别插件。

此分支为精简定制版本，主要移除了模型规格的手动切换卡片以精简配置流程，并针对多语言（韩语）的离线识别进行了重定向适配。

## ⚙️ 核心修改说明

1. **多语言识别逻辑调整**
   - **自动识别（Auto）智能预判**：当源语言设为“自动识别”时，插件在后台首先对图片正中央 100 像素高度的窄条区域进行轻量级的双模型识别比对，对比有效字符识别数量以确定语种，随后使用正确的模型对原图执行单次完整推理，以实现自动识别模式下对韩语的支持。
   - **韩语重定向**：当明确选择韩语 (`LangEnum.Korean`) 时，自动加载专用的 `KoreanV4` 离线模型。
   - **其他多语言**：简体中文、繁体中文、英文、日语等其余语言加载 `PP-OCRv6` 统一多语言模型（Small 规格）。

2. **简化配置项**
   - 移除了设置面板中关于 `Tiny / Small / Medium` 规格的手动切换组件。
   - 后台默认锁定为性能与精度较为均衡的 **Small 规格** 模型，简化用户的配置流程。

3. **云端自动构建**
   - 增加了 GitHub Actions 工作流，支持在推送 Tag 时自动编译并发布包含 `.spkg` 格式插件包的 Release。
